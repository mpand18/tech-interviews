# Tech Interviews – Backend, Mobile & Leadership

Este repositorio contiene un banco de challenges cortos para entrevistas técnicas
de roles de **Backend**, **Mobile**.

Los ejercicios son de **diseño y razonamiento**, no de implementación completa:
no hace falta que el código corra. Cada challenge está pensado para
**45–60 minutos de entrevista**.

## Estructura

- `guides/`
  - Cómo usar los challenges
  - Rúbricas por nivel
- `challenges/backend/`
- `challenges/mobile/`

## Cómo usarlo

1. Elegí el challenge según stack (backend / mobile ) y seniority.
2. Compartí el `.md` con la persona (link, pantalla o contenido).
3. Pedile que piense en voz alta, dibuje, modele y explique sus decisiones.
4. Usá la sección **Criterios de evaluación** del challenge + las rúbricas de `guides/`
   para tomar notas y decidir.
5. No busques soluciones perfectas: el objetivo es ver **cómo piensan**, no que implementen una app de producción en 1 hora.
